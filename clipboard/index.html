ClipboardClipboard

Install

// index.js
import ClipboardController from '@viget/stimulus-controllers/clipboard_controller'
Stimulus.register('clipboard', ClipboardController)

Values

ValueFull AttributeTypeDefaultDescription
Success Delaydata-clipboard-success-delay-valueNumber1000 (ms)Time (ms) value passed to setTimeout() that removes toggleSuccess class after copy event. If either ctaText or successText target is missing, this does nothing except emit success-dismissed event after specified time.

Targets

TargetFull AttributeDescriptionRequired
sourcedata-clipboard-target="source"Add to the element (<input>, <div>, etc) whose content should be copied. Text is pulled by checking for a value first; otherwise, it uses the innerText. Relies on the Clipboard APIYes
ctaTextdata-clipboard-target="ctaText"Optional hook for toggling between CTA text (e.g. “copy”) and success text (e.g. “copied”). Add to an element showing the CTA message. Must be used in tandem with successText to have any effect.No
successTextdata-clipboard-target="successText"Optional hook for showing success text. Add to an element showing the success message. Must be used with ctaText to have any effect.No

Classes

ClassFull AttributeDescription
toggleSuccessdata-class-list-toggle-success-classClasses that are removed from the ctaText and added to the successText targets upon copy action, if both targets exist. After the successDelay ms value, these reset.

Actions

copy()

Adds text from source target to user’s clipboard. Text is pulled by checking for an element value first; otherwise it uses the innerText.

EventDescription
clipboard:copyEmitted immediately
clipboard:success-dismissedEmitted with setTimeout() using successDelay value

Demos

Copy from input

<div data-controller="clipboard">
  <input
    type="text"
    value="Input text to be copied"
    data-clipboard-target="source"
  >
  <button class="relative" data-action="clipboard#copy">
    <span data-clipboard-target="ctaText">Copy</span>
    <span
      class="opacity-0 invisible absolute inset-0 flex items-center justify-center"
      data-clipboard-target="successText"
    >
      Copied!
    </span>
  </button>
</div>
